‚ùì Troubleshooting Settings

Cannot access certain settings categories

This is likely a privilege issue. Contact your administrator to verify your user permissions and access levels.

Database connection test fails

Check these common issues:

  • Verify server address and port settings are correct
  • Ensure database server is running and accessible
  • Check username and password credentials
  • Verify network connectivity to database server

Theme changes don't apply immediately

Some theme changes require application restart to take full effect. Try closing and reopening the application to see all theme updates.

Settings changes are lost after restart

This indicates a permission or configuration issue:

  • Ensure you clicked "Save" before closing Settings form
  • Check database write permissions for your user account
  • Verify the application can write to the user settings database table
  • Contact IT support if problem persists
