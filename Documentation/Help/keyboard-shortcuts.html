Customizing Shortcuts

‚öôÔ∏è Personalization

Some shortcuts can be customized through the Settings form:

  1. Open File ‚Üí Settings (or press Ctrl+,)
  2. Navigate to the Shortcuts section
  3. Select the shortcut you want to modify
  4. Press the new key combination
  5. Check for conflicts and resolve them
  6. Click Save to apply changes
Note: Some system shortcuts (F1, Alt+F4, etc.) cannot be modified as they are reserved by the operating system.
