‚ùì Troubleshooting Advanced Features

Excel import fails with format error

Common solutions:

  • Download and use the official Excel template
  • Ensure column headers exactly match template requirements
  • Check for special characters or formatting issues in data
  • Verify Excel file is saved as .xlsx format

Multi-location entry creates duplicate items

This is expected behavior when using multi-location mode. Each location gets a separate inventory record with the specified quantity.

Batch operation runs very slowly

Performance optimization steps:

  • Reduce batch size in advanced settings
  • Close other applications to free system resources
  • Check database server performance and connectivity
  • Consider processing during off-peak hours

Advanced Remove filters return no results

Check filter settings:

  • Verify date range includes expected transactions
  • Check user filter includes correct usernames
  • Try clearing all filters and start with basic criteria
  • Ensure you have appropriate permissions for filtered data
