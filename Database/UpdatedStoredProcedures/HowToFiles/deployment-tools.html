🔧 Windows Deployment Tools

The MTM Inventory Application provides 6 specialized batch files for consolidated deployment management:

🚀 deploy.bat
Consolidated deployment script
  • Tests database connection
  • Creates backup of existing procedures
  • Deploys mtm_wip_application_test_routines.sql
  • Shows total procedure count (~96 procedures)
# Basic usage
deploy.bat

# With parameters
deploy.bat -h localhost -u root -p root -d mtm_wip_application_test
🔍 analyze.bat
Consolidated file analysis tool
  • Analyzes consolidated SQL file
  • Counts procedures by category
  • Verifies p_ parameter naming
  • Checks MySQL 5.7.24 compatibility
  • Validates DELIMITER $$ syntax
# Run analysis on consolidated file
analyze.bat
✅ run_verification.bat
Consolidated system verification
  • Verifies consolidated procedures deployment
  • Tests key procedure categories
  • Checks procedure functionality
  • Provides detailed status report
# Verify consolidated deployment
run_verification.bat
🏁 final_verification.bat
Quick consolidated status check
  • Fast execution for consolidated file
  • Tests procedure categories
  • Shows overall system status
  • Provides pass/fail summary
# Quick consolidated system health check
final_verification.bat
🔧 fix_and_reverify.bat
Schema repair for consolidated deployment
  • Creates missing tables for consolidated routines
  • Re-deploys consolidated SQL file
  • Adds sample data where needed
  • Re-runs complete verification
# Fix schema issues and re-deploy consolidated file
fix_and_reverify.bat
🔍 find_missing_procedures.bat
Missing procedures detection & consolidation tool
  • Scans UpdatedStoredProcedures\ directory for missing items
  • Compares individual SQL files against consolidated file
  • Automatically adds missing procedures to consolidated file
  • Generates detailed missing items report
# Find and add missing procedures
find_missing_procedures.bat

🆕 NEW Tool

This tool searches for procedures, functions, and triggers that exist in individual SQL files but are missing from the consolidated file, and can automatically add them.

🔄 Tool Usage Workflow

1. Detection Phase

Use find_missing_procedures.bat to identify any procedures missing from the consolidated file

2. Analysis Phase

Run analyze.bat to check code quality and compatibility of the consolidated file

3. Deployment Phase

Deploy with deploy.bat to install all procedures to your database

4. Verification Phase

Verify deployment with run_verification.bat for comprehensive testing

5. Quick Check

Use final_verification.bat for fast status checks during development

6. Repair & Fix

If issues arise, use fix_and_reverify.bat to repair and re-test
