⚡ Deploy in 3 Simple Steps

MAMP users can deploy all procedures in under 5 minutes with default settings

🚀 Quick Start Guide

Ensure MAMP is Running

  • Start MAMP application
  • Verify Apache and MySQL services are running (green lights)
  • Note the MySQL port (usually 3306, sometimes 8889 for older MAMP)

💡 MAMP Setup Tip

Default MAMP credentials are usually root/root on port 3306

Deploy Consolidated Procedures

# Navigate to the directory
cd "C:\Users\[YourUsername]\source\repos\MTM_WIP_Application\Database\UpdatedStoredProcedures"

# Deploy consolidated procedures (MAMP defaults)
deploy.bat

# Or with custom parameters
deploy.bat -h localhost -u root -p root -d mtm_wip_application_test

✅ Expected Output

[SUCCESS] Consolidated stored procedures deployed successfully!

Procedures in file: ~96

Verify Consolidated Deployment

# Run verification
run_verification.bat

# Or quick final check
final_verification.bat

✅ Verification Success

✅ CONSOLIDATED ROUTINES VERIFIED AND WORKING!

💡 Environment-Specific Usage

Development Environment (Recommended)

# Deploy to test database
deploy.bat -d mtm_wip_application_test

# Analyze and verify
analyze.bat
run_verification.bat

Production Environment

# Deploy to production
deploy.bat -d mtm_wip_application -h 172.16.1.104

# Verify production
final_verification.bat

Local MAMP Development (Most Common)

# Standard MAMP deployment
deploy.bat

# Verify MAMP deployment
run_verification.bat
